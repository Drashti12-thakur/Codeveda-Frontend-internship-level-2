# Codeveda-Frontend-internship-level-2
# 🌐 Frontend Development – Level 2 Project

This project showcases advanced frontend development skills including API integration, responsive UI design, and dynamic data rendering.

---

## 📌 Project Overview

The project consists of two main features:

1. 🌦️ Weather Application  
2. 🧑‍💻 GitHub Profile Finder  

Both features use external APIs to fetch real-time data and display it in a clean and responsive interface.

---

## 🌦️ Weather Application

A dynamic weather app that fetches real-time weather data based on user input.

### ✨ Features
- Search weather by city name  
- Real-time temperature and conditions  
- Dynamic UI updates  
- Error handling for invalid inputs  

### 🔗 API Used
- OpenWeather API

---

## 🧑‍💻 GitHub Profile Finder

Fetches and displays GitHub user profile data.

### ✨ Features
- Search GitHub users by username  
- Display profile details (name, bio, repos, followers)  
- Profile image and links  
- Error handling for invalid usernames  

### 🔗 API Used
- GitHub Users API

---

## 🎨 CSS Framework

The project uses a CSS framework for building a responsive and modern UI.

### ✨ Features

---

## 🛠️ Tech Stack

- HTML

## ▶️ How to Run

1. Clone the repository  
2. Open the project folder  
3. Run `index.html` in browser  
4. Ensure internet connection for API requests
